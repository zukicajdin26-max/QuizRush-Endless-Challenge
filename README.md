# 🎮 QuizRush: Endless Challenge

QuizRush is a fast-paced, browser-based quiz game built with HTML, CSS, and JavaScript. Players must answer questions quickly and accurately to survive as long as possible and achieve the highest score.

## 🎯 Gameplay

* Answer randomly generated questions
* Each correct answer increases your score
* Build a streak to earn bonus points
* Time pressure makes the game more challenging
* One wrong answer = Game Over
